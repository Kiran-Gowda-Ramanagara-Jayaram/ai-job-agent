AI Job Agent

Semi-autonomous job discovery, artifact generation, recruiter outreach, and online learning. Built by Kiran Gowda Ramanagara Jayaram.

Read the README View the Streamlit app code Run locally Results

RAG-grounded artifacts

Resume bullets, cover letter, prep pack, and 20 STAR answers conditioned on JD/company snippets.

RoleFit model

Cosine features + logistic regression estimate $p(y{=}1\\mid x)$ to rank jobs by fit.

Bandit-driven outreach

Thompson sampling over templates per bucket (role|company) learns what works.

Key math

Logistic regression

$$\Pr(y=1\mid x)=\sigma(w^\top x+b),\qquad \sigma(z)=\frac{1}{1+e^{-z}}$$
$$\min_{w,b}\ \sum_{i=1}^{n}\Big[-y_i\ln p_i-(1-y_i)\ln(1-p_i)\Big]+\lambda\lVert w\rVert_2^2,\quad p_i=\sigma(w^\top x_i+b)$$

RAG retrieval

$$\operatorname*{arg\,topk}_{m\in\{1,\ldots,M\}}\ \cos\!\big(\phi(q),\,\phi(d_m)\big)$$

$\phi(\cdot)$ is the embedding function; $\cos$ is cosine similarity.

Bandit posterior (Beta–Bernoulli TS)

$$\alpha \leftarrow \alpha + r,\qquad \beta \leftarrow \beta + 1 - r$$
$$\tilde{\theta}\sim \mathrm{Beta}(\alpha,\beta),\qquad k^\*=\arg\max_k \tilde{\theta}_k$$

Run locally

# 1) Setup
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2) Configure
# .env with OPENAI_API_KEY and HUNTER_API_KEY
# data/profile.yaml with roles, locations, keywords

# 3) Ingest jobs and clean up
python -m scripts.ingest_presets --limit 100
python -m scripts.cleanup_jobs

# 4) (Optional) Train RoleFit
python -m scripts.seed_synthetic && python -m scripts.train_fit_model

# 5) Launch
streamlit run app/app.py
