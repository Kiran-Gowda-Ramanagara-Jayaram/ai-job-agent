Semi-autonomous job discovery, artifact generation, recruiter outreach, and online learning. Built by Kiran Gowda Ramanagara Jayaram.

Highlights

RAG-grounded artifacts; RoleFit (cosine features + logistic regression); Bandit-driven outreach with Thompson sampling.

Architecture

flowchart LR
  A[Ingest scripts\n(Greenhouse, Lever, Ashby)] --> B[(SQLite jobs.db)]
  B --> C[RoleFit v2\n(cosine + logistic)]
  C --> D[Streamlit App\n(Queue & Dashboard)]
  D --> E[Artifact Generation (LLM)\nResume • Cover • STAR • Prep]
  D --> F[RAG store\n(SimpleStore)]
  D --> G[Hunter.io\nDomain & Name finder]
  D --> H[(CRM: contacts & outreach)]
  H --> I[Bandit learner\n(Thompson sampling)]
  I --> D

Key math

Logistic regression

$$\Pr(y=1\mid x)=\sigma(w^\top x+b),\qquad \sigma(z)=\frac{1}{1+e^{-z}}.$$
$$\min_{w,b}\ \sum_{i=1}^n\Big[-y_i\log p_i-(1-y_i)\log(1-p_i)\Big]+\lambda\lVert w\rVert_2^2,\ \ p_i=\sigma(w^\top x_i+b).$$

RAG retrieval

$$\operatorname*{arg\,topk}_{m}\ \cos\!\big(\phi(q),\,\phi(d_m)\big),$$

where $\phi(\cdot)$ is the embedding function and $\cos$ is cosine similarity.

Bandit posterior

$$(\alpha,\beta)\leftarrow(\alpha+r,\ \beta+1-r),\qquad \tilde\theta\sim\mathrm{Beta}(\alpha,\beta),\ \ k^\*=\arg\max_k \tilde\theta_k.$$

Run locally

# 1) Setup
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2) Configure
# .env with OPENAI_API_KEY and HUNTER_API_KEY
# data/profile.yaml with roles, locations, keywords

# 3) Ingest jobs and clean up
python -m scripts.ingest_presets --limit 100
python -m scripts.cleanup_jobs

# 4) (Optional) Train RoleFit
python -m scripts.seed_synthetic && python -m scripts.train_fit_model

# 5) Launch
streamlit run app/app.py
